# 如何使用

每一集视频，对应一个代码分支。

比如你现在收看的视频是： 01-springboot的hello world

那么你只需要 切换到 对应的分支就可以了。

```git
git checkout branch_01 
```
这样一来大家对于某一集的内容如果有什么不明白的

都可以 找到对应的代码 ，从而进行调试和交流

避免 不必要的 时间开销。
